**<font style="color:rgb(0, 0, 0);background-color:rgb(254, 254, 254);">2. 心电图：</font>**<font style="color:rgb(0, 0, 0);background-color:rgb(254, 254, 254);">当心脏因缺血受损或坏死时，心电活动也会出现异常，并以放大的</font>**<font style="color:rgb(0, 0, 0);background-color:rgb(254, 254, 254);">“波形”</font>**<font style="color:rgb(0, 0, 0);background-color:rgb(254, 254, 254);">直观地反映在心电图上。据此可以诊断是否存在</font><font style="color:rgb(0, 0, 0);background-color:rgb(254, 254, 254);">心律失常</font><font style="color:rgb(0, 0, 0);background-color:rgb(254, 254, 254);">、</font><font style="color:rgb(14, 64, 140);background-color:rgb(254, 254, 254);">心肌缺血</font><font style="color:rgb(0, 0, 0);background-color:rgb(254, 254, 254);">、心肌梗死和心肌肥厚等病症。</font>

**<font style="color:rgb(0, 0, 0);background-color:rgb(254, 254, 254);">3.</font>****<font style="color:rgb(14, 64, 140);background-color:rgb(254, 254, 254);">24小时动态心电图</font>****<font style="color:rgb(0, 0, 0);background-color:rgb(254, 254, 254);">：</font>**<font style="color:rgb(0, 0, 0);background-color:rgb(254, 254, 254);">也就是</font>**<font style="color:rgb(0, 0, 0);background-color:rgb(254, 254, 254);"> </font>****<font style="color:rgb(0, 0, 0);background-color:rgb(254, 254, 254);">“Holter”</font>**<font style="color:rgb(0, 0, 0);background-color:rgb(254, 254, 254);">。与普通的静态心电图相比，24小时动态心电图能够持续监测患者的心电活动，帮助发现那些平时隐蔽的心律失常和无症状心肌缺血等病症。</font>

**<font style="color:rgb(0, 0, 0);background-color:rgb(254, 254, 254);">4.24小时动态血压：</font>**<font style="color:rgb(0, 0, 0);background-color:rgb(254, 254, 254);">血压</font><font style="color:rgb(0, 0, 0);background-color:rgb(254, 254, 254);">可以说无时无刻不在变化着，24小时动态血压可以掌握一天之内全面的血压变化情况。该项目有助于筛查</font><font style="color:rgb(14, 64, 140);background-color:rgb(254, 254, 254);">隐蔽性高血压</font><font style="color:rgb(0, 0, 0);background-color:rgb(254, 254, 254);">和夜间高血压，也有助于高血压的分级、判断高血压的昼夜节律类型以及评价降压药的效果等。</font>

<font style="color:rgb(51, 51, 51);background-color:rgb(254, 254, 254);">  
</font>

**<font style="color:rgb(0, 0, 0);background-color:rgb(254, 254, 254);">5.运动平板试验：</font>**<font style="color:rgb(0, 0, 0);background-color:rgb(254, 254, 254);">又叫</font>**<font style="color:rgb(0, 0, 0);background-color:rgb(254, 254, 254);">“运动负荷试验”</font>**<font style="color:rgb(0, 0, 0);background-color:rgb(254, 254, 254);">。有些</font><font style="color:rgb(0, 0, 0);background-color:rgb(254, 254, 254);">冠心病</font><font style="color:rgb(0, 0, 0);background-color:rgb(254, 254, 254);">患者在安静状态下跟健康人一样正常，只有</font>**<font style="color:rgb(0, 0, 0);background-color:rgb(254, 254, 254);">增加心脏负荷，提高心肌耗氧量</font>**<font style="color:rgb(0, 0, 0);background-color:rgb(254, 254, 254);">，才能激发出潜在的病变。检查时让受检者背着相关监护仪器，在跑步机上由走到跑，慢慢增加活动强度，然后观察这一过程中心电图和血压的变化。</font>**<font style="color:rgb(95, 156, 239);background-color:rgb(254, 254, 254);">运动平板试验也经常用来指导健康人和心脏病患者的安全运动</font>**<font style="color:rgb(0, 0, 0);background-color:rgb(254, 254, 254);">。</font>

<font style="color:rgb(51, 51, 51);background-color:rgb(254, 254, 254);">  
</font>

**<font style="color:rgb(0, 0, 0);background-color:rgb(254, 254, 254);">6.心脏彩超：</font>**<font style="color:rgb(0, 0, 0);background-color:rgb(254, 254, 254);">相当于医生的</font>**<font style="color:rgb(0, 0, 0);background-color:rgb(254, 254, 254);">“透视眼”</font>**<font style="color:rgb(0, 0, 0);background-color:rgb(254, 254, 254);">，能够动态显示心脏结构是否坚固、瓣膜关闭是否严密、心肌收缩舒张是否正常、血液流动是否畅通等。</font>

**<font style="color:rgb(0, 0, 0);background-color:rgb(254, 254, 254);">8.冠脉CT：</font>**<font style="color:rgb(0, 0, 0);background-color:rgb(254, 254, 254);">通过CT成像可以</font>**<font style="color:rgb(0, 0, 0);background-color:rgb(254, 254, 254);">模拟出患者的冠脉系</font>**<font style="color:rgb(0, 0, 0);background-color:rgb(254, 254, 254);">统，因此能够观察血管是否有淤堵或破裂。冠脉CT还能够测量患者冠脉狭窄、钙化斑块和</font><font style="color:rgb(0, 0, 0);background-color:rgb(254, 254, 254);">冠脉</font><font style="color:rgb(0, 0, 0);background-color:rgb(254, 254, 254);">管壁等具体情况。当我们怀疑患者有冠心病，冠脉CT检查是重要选项。若检查结果正常，基本不需要进一步检查。</font>
