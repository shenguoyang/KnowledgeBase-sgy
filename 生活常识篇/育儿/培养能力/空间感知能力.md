# **什么是空间思维能力**
主要包括了四个方面

+ 对方向定位的知觉（如空间的上、下、前、后、左、右方向的定位）；
+ 对距离的知觉（比如对物体位置的远近判断）
+ 对图形辨认的知觉（参阅我们前期关于认知图形的文章）
+ 对图形之间的关系和图形变化结果的洞察和直觉

**前两者就是空间感知，孩子对空间的感知能力不是与生俱来的，而是在生活经验当中逐步发展起来的。因此，发展空间感知能力需要多给孩子探索的机会，并加以合适的引导。**

它包含一般的视力、视觉聚焦能力、颜色辨别能力、轮廓掌握能力、视野扩展能力、平面与立体的转换能力、图像创造能力和层次辨别能力等。

# **幼儿空间思维的发展规律**
**幼儿空间思维能力的发展主要是从以自我为中心的空间意识阶段（以自己判断其他物体的空间方位）走向以客体为中心的空间意识阶段（逐渐掌握相对性）。**

不同年龄的幼儿发展规律如下：

**小班：**

1. **能判断物体之间明显的空间关系（主要是上下里外，如一杯水在桌子上）。**
2. **能区分和说出以自身为中心的上下前后方位，理解里面和外面的概念。**

**中班：**

1. 能区分和说出以自身为中心的前后左右远近方位。
2. 能说出物体之间的空间关系（关键在于表达出来）。

**大班**

1. 能以自身为中心和以客体为中心区分左右。
2. 会向左、向右方向运动。
3. 能注意自己（或物体）在空间的位置和运动方向。

# **如何引导小班幼儿发展空间思维能力**
对小班幼儿空间认知能力发展的目标要求是：能够区分以自我为中心的上下、前后方位，能够判断两个物体之间明显的上下关系，并清楚表达出来。

**1、首先帮助幼儿理解上下 、前后方位词的意义**

**正确理解方位词是幼儿认识空间方位关系的基础。**

小班幼儿的思维特点是具体形象思维为主，因此家长在进行空间方位教学时，应该用幼儿熟悉的、真实可见的物体来进行空间方位描述。**例如摸摸你的小脑袋，这是上 ；敲敲你的小脚丫，这是下。**

**2、充分利用生活环境，抓住教育契机**

幼儿空间认知的发展是在与环境相互作用的过程中完成的，家长需要给孩子更多的空间探索机会并加以影刀。

**比如家长可以引导幼儿自己取放物品，按照指令整理物品“把袜子放在裤子上”“把蓝色的积木放在红色的积木后面”**

![](https://cdn.nlark.com/yuque/0/2023/jpeg/26460094/1679738706571-c8099f30-cfc9-4e68-ba81-c5b122442dea.jpeg)

**3、营造环境 ，引导幼儿大胆地描述物体之间的关系**

家长要创设一个让孩子能说 、想说、敢说的氛围。例如，在生活当中让孩子帮忙找东西，请孩子说一说东西在哪里，不要用手指。表达时要求准确表达，例如说出什么物体在上，什么物体在下。

# **空间小游戏**
**1、走迷宫**

走迷宫游戏是培养孩子空间能力的一种非常好的方式。当孩子进行走迷宫游戏时，研究者发现孩子大脑里记录空间关系的顶叶区变得活跃。

![](https://cdn.nlark.com/yuque/0/2023/jpeg/26460094/1679738802873-17bbda23-14c4-4720-b682-a5ecdb34458a.jpeg)

**2、找东西**

爸爸、妈妈把东西放在某处，让孩子去找一找，并用方位词讲述。如帽子藏在在枕头下面、围巾在沙发上等，逐一进行，让孩子自己找到自己的东西，这样可训练孩子的空间方位知觉，并真正理解表达方位的词语。或者可以反过来爸爸妈妈描述东西藏在哪里，让孩子根据提示去找寻。

![](https://cdn.nlark.com/yuque/0/2023/png/26460094/1679738802904-ffbf6f3f-ec10-41ce-89fe-b06729041950.png)

**3、拼图**

拼图在空间上的应用其实是整体和部分，有的孩子虽然知道拼版的上下左右位置，但是还是无法通过旋转图案来完成拼接，家长一开始可以选择拼块少一些的，比如4块，等孩子熟练了，再增加数量。

![](https://cdn.nlark.com/yuque/0/2023/jpeg/26460094/1679738802898-a1841fbd-7218-451d-9c33-1cbc8cac0441.jpeg)

**4、找位置**

![](https://cdn.nlark.com/yuque/0/2023/png/26460094/1679738802865-f25e7cdd-7fe5-409b-a6fe-c2a8f0cf1b95.png)

**5、运动游戏当中渗透方向**

“向上举”“向下蹲”“向前走”“向后退”“向右转”等，带领孩子户外障碍运动。也可以玩一玩捉迷藏的游戏，可以让孩子在找到同伴是说一说是在何处找到的，巩固孩子的方位概念。

**6、美术手工活动**

让孩子学习画画、折纸等表现物体前后关系，上下关系等。

**8.记忆宫殿**

# **空间概念绘本**
**1、《我知道上下左右啦》**

小动物们和老师一起去森林里春游，老师为每个小动物都准备了一份礼物，但是它们必须先找到一件宝贝才能得到这个礼物。小鸭子呱呱想第一个找到礼物，但是它不能到树上去找，也不能到地下去找，它怎么才能找到它的宝贝呢？小呱呱的宝贝只有它才能找到哦。

**小朋友们在刚刚接触方位的时候，往往会混淆这些方位。这个故事通过小动物们的寻宝过程，以及每个小动物的特点，巧妙地把方位词贯穿其中**。

![](https://cdn.nlark.com/yuque/0/2023/png/26460094/1679738802951-bd2aba82-706d-4872-bc61-42b26bd2ad81.png)

**2、《神奇的玩具箱》**

尼基是个聪明活泼的小男孩。他有一个装玩具的箱子，他常常坐在他的箱子里去“旅行”。因为他的箱子可以变成小汽车去兜风，可以变成飞机飞上天，还可以变成军舰和潜水艇，甚至可以变成太空飞船。当然，这些都是尼基的想象。

**通过这些“旅行”，小朋友们就可以跟着尼基熟悉前后里外等空间概念。父母也可以根据小朋友的熟知程度，帮助他们理解“左上，左下，右上，右下”等扩展方位。**
