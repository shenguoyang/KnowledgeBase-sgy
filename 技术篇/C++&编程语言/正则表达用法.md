# 1. 正则表达规则
## 1.1 字符
+ x  		表示字符x
+ \\ 		表示反斜线字符
+ \n 		表示换行
+ \r 		表示回车

## 1.2 字符类
+ [abc] 	 表示一个字符，a、b 或 c
+ [^abc]    表示任何一个字符，除了a、b 或 c
+ [a-zA-Z] 表示一个字符，a到z或者A到Z
+ [0-9] 	 表示一个字符，0到9

举例：

1）校验一个字符串中是否含有字母a或b

"[ab]"


## 1.3 预定义字符
+ . 任何字符，\\. 表示 . 本身
+ \d 数字：[0,9]
+ \D 表示不是数字字符
+ \w 单词字符：[a-zA-Z_0-9]
+ \W 表示不是字母、数字、下划线

注意，C++中要用双斜杠

## 1.4 数量词
+ X?   	表示X出现**0次**或者**1次**
+ X*   		表示X出现**0次**或者**1次**或者**多次**
+ X+ 		表示X出现**1次**或者**多次**
+ X{n} 	表示X**恰好**出现n次
+ X{n, } X**至少**出现n次
+ X{n,m} X**至少**出现n次，但**不超过**m次

举例：

1）校验密码的合法性（假设密码只能由8位以上的数字或字符组成）

"\\w{8,}"

## 1.5 边界匹配器
+ ^ 行的开头
+ $ 行的结尾
+ \b 单词边界

举例：

1）校验电话号码合法性（假设电话号码只能由11位数字组成）

"^\\d[11]$"

# 2. C++组件库使用
## 2.1 regex_match
1）全文匹配函数regex_match，返回值bool类型，表示是否匹配成功

2）字符串与正则表达**完全**匹配，则返回true，否则返回false

```cpp
#include <regex>
#include <iostream>
using namespace std;
void main()
{
    string  str = "abc123";
    regex pat1("abc");
    bool flag1 = regex_match(str, pat1);//返回fasle  
    regex pat2("abc123");
    bool flag2 = regex_match(str, pat2);//返回true  
    regex pat3("\\w{6}");
    bool flag3 = regex_match(str, pat3);//返回true  
    regex pat4("\\w{5}");
    bool flag4 = regex_match(str, pat4);//返回false  
}
```

## 2.2 regex_search
1）搜索匹配函数regex_search，返回值bool，表示是否匹配成功

2）字符串与正则表达**部分**匹配，则返回true，否则返回false

```cpp
#include <regex>
#include <iostream>
using namespace std;
void main()
{
    string  str = "abc123";
    regex pat1("abc");
    bool flag1 = regex_search(str, pat1);	//返回true 
    regex pat2("1234");
    bool flag2 = regex_search(str, pat2);	//返回fasle  
    regex pat3("ABC");		
    bool flag3 = regex_search(str, pat3);	//返回fasle  
    regex pat4("ABC",regex::icase);			//匹配时忽略大小写，默认不忽略
    bool flag4 = regex_search(str, pat4);	//返回true  

    std::smatch ret; //存储匹配的结果
    if (std::regex_search(str, ret, pat4)
}
```

## 2.3 regex_replace
1）替换匹配函数regex_replace，返回值为替换后的字符串

```cpp
#include <regex>
#include <iostream>
using namespace std;
void main()
{
    string  str = "Hello_2019!";
    regex pat1("(.{3})_(\\d{4})");
    string strReuslt = regex_replace(str, pat1,"");	  //返回He!
}
```

## 2.4 其他
1）结果容器类smatch，存放搜索的结果

2）迭代器

```cpp
#include <regex>
#include <iostream>
using namespace std;

int main()
{
    //容器类smatch，存放搜索的结果
    {
        string  str = "sgy123sgy";
        smatch results;    
        regex pat1("sgy|123");
        if (regex_search(str, results, pat1))
        {
            cout << results[0] << endl;       //输出第一个正则匹配成功的
            cout << results.str() << endl;
        }
    }
    
    //利用迭代器查看所有匹配结果值
    {
        string  str = "sgy123sgy";
        smatch results;
        string::const_iterator iterStart = str.begin();
        string::const_iterator iterEnd = str.end();
        regex pat1("sgy|123");
        while (regex_search(iterStart, iterEnd, results, pat1))
        {
            cout << results[0] << endl;       //输出第一个正则匹配成功的
            cout << results.str() << endl;
            iterStart = results[0].second;
        }
    }
}


```
