# 1. 正则表达规则
## <font style="color:rgb(0, 0, 0) !important;">1.1 字符</font>
+ <font style="color:rgb(34, 34, 34) !important;">x  		表示字符x</font>
+ <font style="color:rgb(34, 34, 34) !important;">\\ 		表示反斜线字符</font>
+ <font style="color:rgb(34, 34, 34) !important;">\n 		表示换行</font>
+ <font style="color:rgb(34, 34, 34) !important;">\r 		表示回车</font>

## <font style="color:rgb(0, 0, 0) !important;">1.2 字符类</font>
+ <font style="color:rgb(34, 34, 34) !important;">[abc] 	 表示一个字符，a、b 或 c</font>
+ <font style="color:rgb(34, 34, 34) !important;">[^abc]    表示任何一个字符，除了a、b 或 c</font>
+ <font style="color:rgb(34, 34, 34) !important;">[a-zA-Z] 表示一个字符，a到z或者A到Z</font>
+ <font style="color:rgb(34, 34, 34) !important;">[0-9] 	 表示一个字符，0到9</font>

<font style="color:rgb(34, 34, 34) !important;">举例：</font>

<font style="color:rgb(34, 34, 34) !important;">1）校验一个字符串中是否含有字母a或b</font>

<font style="color:rgb(34, 34, 34) !important;">"[ab]"</font>

<font style="color:rgb(34, 34, 34) !important;"></font>

## <font style="color:rgb(0, 0, 0) !important;">1.3 预定义字符</font>
+ <font style="color:rgb(34, 34, 34) !important;">. 任何字符，\\. 表示 . 本身</font>
+ <font style="color:rgb(34, 34, 34) !important;">\d 数字：[0,9]</font>
+ \D 表示不是数字字符
+ <font style="color:rgb(34, 34, 34) !important;">\w 单词字符：[a-zA-Z_0-9]</font>
+ \W 表示不是字母、数字、下划线

注意，C++中要用双斜杠

## <font style="color:rgb(0, 0, 0) !important;">1.4 数量词</font>
+ <font style="color:rgb(34, 34, 34) !important;">X?   	表示X出现</font>**<font style="color:rgb(34, 34, 34) !important;">0次</font>**<font style="color:rgb(34, 34, 34) !important;">或者</font>**<font style="color:rgb(34, 34, 34) !important;">1次</font>**
+ <font style="color:rgb(34, 34, 34) !important;">X*   		表示X出现</font>**<font style="color:rgb(34, 34, 34) !important;">0次</font>**<font style="color:rgb(34, 34, 34) !important;">或者</font>**<font style="color:rgb(34, 34, 34) !important;">1次</font>**<font style="color:rgb(34, 34, 34) !important;">或者</font>**<font style="color:rgb(34, 34, 34) !important;">多次</font>**
+ <font style="color:rgb(34, 34, 34) !important;">X+ 		表示X出现</font>**<font style="color:rgb(34, 34, 34) !important;">1次</font>**<font style="color:rgb(34, 34, 34) !important;">或者</font>**<font style="color:rgb(34, 34, 34) !important;">多次</font>**
+ <font style="color:rgb(34, 34, 34) !important;">X{n} 	表示X</font>**<font style="color:rgb(34, 34, 34) !important;">恰好</font>**<font style="color:rgb(34, 34, 34) !important;">出现n次</font>
+ <font style="color:rgb(34, 34, 34) !important;">X{n, } X</font>**<font style="color:rgb(34, 34, 34) !important;">至少</font>**<font style="color:rgb(34, 34, 34) !important;">出现n次</font>

<font style="color:rgb(34, 34, 34) !important;">举例：</font>

## <font style="color:rgb(0, 0, 0) !important;">1.5 边界匹配器</font>

<font style="color:rgb(34, 34, 34) !important;">举例：</font>

# 2. C++组件库使用
## 2.1 regex_match
1）全文匹配函数regex_match，返回值bool类型，表示是否匹配成功

2）字符串与正则表达**完全**匹配，则返回true，否则返回false

```cpp
#include <regex>
#include <iostream>
using namespace std;
void main()
{
    string  str = "abc123";
    regex pat1("abc");
    bool flag1 = regex_match(str, pat1);//返回fasle  
    regex pat2("abc123");
    bool flag2 = regex_match(str, pat2);//返回true  
    regex pat3("\\w{6}");
    bool flag3 = regex_match(str, pat3);//返回true  
    regex pat4("\\w{5}");
    bool flag4 = regex_match(str, pat4);//返回false  
}
```

## 2.4 其他
1）结果容器类smatch，存放搜索的结果

2）迭代器

```cpp
#include <regex>
#include <iostream>
using namespace std;

int main()
{
    //容器类smatch，存放搜索的结果
    {
        string  str = "sgy123sgy";
        smatch results;    
        regex pat1("sgy|123");
        if (regex_search(str, results, pat1))
        {
            cout << results[0] << endl;       //输出第一个正则匹配成功的
            cout << results.str() << endl;
        }
    }
    
    //利用迭代器查看所有匹配结果值
    {
        string  str = "sgy123sgy";
        smatch results;
        string::const_iterator iterStart = str.begin();
        string::const_iterator iterEnd = str.end();
        regex pat1("sgy|123");
        while (regex_search(iterStart, iterEnd, results, pat1))
        {
            cout << results[0] << endl;       //输出第一个正则匹配成功的
            cout << results.str() << endl;
            iterStart = results[0].second;
        }
    }
}


```
