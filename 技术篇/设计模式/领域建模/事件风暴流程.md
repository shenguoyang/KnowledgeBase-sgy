# DDD事件风暴（Event Storming）详解
事件风暴是DDD中最核心的协作建模技术，我通过一个**完整的电商订单系统**实例来详细展示。

## 一、什么是事件风暴？
```plain
事件风暴 = 协作工作坊 + 可视化建模 + 业务流程发现

目标：让业务专家和技术专家用同一种语言理解业务
输出：识别出领域事件、命令、聚合、策略等核心领域元素
```

## 二、事件风暴的3个阶段
```plain
1. 大爆炸探索（Big Picture）- 全景视图
2. 流程建模（Process Modeling）- 详细流程  
3. 软件设计（Software Design）- 技术实现
```

#### 3.2 设计聚合的C++代码
```cpp
// 基于事件风暴结果设计的聚合代码
#pragma once
#include <string>
#include <vector>
#include <chrono>
#include <memory>

namespace domain::order {

// 值对象：订单状态（从事件识别）
enum class OrderStatus {
    CREATED,      // 订单已创建
    SUBMITTED,    // 订单已提交
    PAID,         // 订单已支付
    CONFIRMED,    // 支付已确认
    INVENTORY_DEDUCTED, // 库存已扣减
    SHIPPED,      // 订单已发货
    DELIVERED,    // 订单已送达
    COMPLETED,    // 订单已完成
    CANCELLED,    // 订单已取消
    REFUNDING,    // 退款处理中
    REFUNDED      // 已退款
};

// 值对象：订单金额（从"订单金额已计算"事件识别）
class OrderAmount {
private:
    double subtotal;      // 商品小计
    double shippingFee;   // 运费
    double discount;      // 折扣
    double total;         // 应付总额
    
public:
    OrderAmount(double subtotal, double shippingFee, double discount)
        : subtotal(subtotal), shippingFee(shippingFee), discount(discount) {
        calculateTotal();
    }
    
    void applyDiscount(double newDiscount) {
        if (newDiscount < 0 || newDiscount > subtotal) {
            throw std::invalid_argument("Invalid discount amount");
        }
        this->discount = newDiscount;
        calculateTotal();
    }
    
    void updateShippingFee(double newFee) {
        if (newFee < 0) {
            throw std::invalid_argument("Shipping fee cannot be negative");
        }
        this->shippingFee = newFee;
        calculateTotal();
    }
    
private:
    void calculateTotal() {
        total = subtotal + shippingFee - discount;
    }
};

// 实体：订单项（从"订单商品已添加"事件识别）
class OrderItem : public Entity {
private:
    std::string productId;
    std::string productName;
    double unitPrice;
    int quantity;
    double itemDiscount;  // 商品级折扣
    
public:
    OrderItem(const std::string& productId, 
              const std::string& productName,
              double unitPrice,
              int quantity)
        : productId(productId), productName(productName),
          unitPrice(unitPrice), quantity(quantity), itemDiscount(0.0) {
        validate();
    }
    
    double calculateSubtotal() const {
        return unitPrice * quantity - itemDiscount;
    }
    
    void updateQuantity(int newQuantity) {
        if (newQuantity <= 0) {
            throw DomainException("Quantity must be positive");
        }
        this->quantity = newQuantity;
    }
    
    void applyItemDiscount(double discount) {
        double maxDiscount = unitPrice * quantity;
        if (discount < 0 || discount > maxDiscount) {
            throw DomainException("Invalid item discount");
        }
        this->itemDiscount = discount;
    }
    
private:
    void validate() const {
        if (productId.empty()) throw ValidationException("Product ID is required");
        if (productName.empty()) throw ValidationException("Product name is required");
        if (unitPrice <= 0) throw ValidationException("Unit price must be positive");
        if (quantity <= 0) throw ValidationException("Quantity must be positive");
    }
};

// 聚合根：订单（核心聚合，从多个事件识别）
class Order : public AggregateRoot {
private:
    std::string orderId;
    std::string customerId;
    OrderStatus status;
    OrderAmount amount;
    std::vector<OrderItem> items;
    std::string shippingAddress;
    std::string paymentMethod;
    std::string paymentTransactionId;
    std::string trackingNumber;
    std::chrono::system_clock::time_point createdAt;
    std::chrono::system_clock::time_point updatedAt;
    
    // 从"订单已取消"事件识别的取消信息
    struct CancellationInfo {
        std::string reason;
        std::chrono::system_clock::time_point cancelledAt;
        bool refundRequired;
    } cancellationInfo;
    
    // 从"退款已处理"事件识别的退款信息
    struct RefundInfo {
        std::string refundId;
        double refundAmount;
        std::string refundReason;
        std::chrono::system_clock::time_point refundedAt;
    } refundInfo;
    
public:
    Order(const std::string& orderId, const std::string& customerId)
        : orderId(orderId), customerId(customerId),
          status(OrderStatus::CREATED),
          amount(0.0, 0.0, 0.0),
          createdAt(std::chrono::system_clock::now()),
          updatedAt(createdAt) {
        validateInitialState();
    }
    
    // ==================== 业务方法（从命令和事件推导） ====================
    
    // 对应命令："添加商品到订单"，事件："订单商品已添加"
    void addItem(const OrderItem& item) {
        if (status != OrderStatus::CREATED) {
            throw DomainException("Cannot add items to order in current status");
        }
        
        items.push_back(item);
        updateOrderAmount();
        updatedAt = std::chrono::system_clock::now();
        
        // 发布领域事件
        addDomainEvent(std::make_shared<OrderItemAddedEvent>(
            orderId, item.getProductId(), item.getQuantity()));
    }
    
    // 对应命令："提交订单"，事件："订单已提交"
    void submit(const std::string& address, const std::string& paymentMethod) {
        if (status != OrderStatus::CREATED) {
            throw DomainException("Order already submitted");
        }
        
        if (items.empty()) {
            throw DomainException("Cannot submit empty order");
        }
        
        validateShippingAddress(address);
        validatePaymentMethod(paymentMethod);
        
        this->shippingAddress = address;
        this->paymentMethod = paymentMethod;
        this->status = OrderStatus::SUBMITTED;
        this->updatedAt = std::chrono::system_clock::now();
        
        addDomainEvent(std::make_shared<OrderSubmittedEvent>(
            orderId, customerId, amount.getTotal(), items.size()));
    }
    
    // 对应命令："支付订单"，事件："订单已支付"
    void markAsPaid(const std::string& transactionId, double paidAmount) {
        if (status != OrderStatus::SUBMITTED) {
            throw DomainException("Order must be submitted before payment");
        }
        
        // 支付金额验证（业务规则）
        if (std::abs(paidAmount - amount.getTotal()) > 0.01) {
            throw DomainException("Paid amount does not match order total");
        }
        
        this->paymentTransactionId = transactionId;
        this->status = OrderStatus::PAID;
        this->updatedAt = std::chrono::system_clock::now();
        
        addDomainEvent(std::make_shared<OrderPaidEvent>(
            orderId, transactionId, paidAmount, paymentMethod));
    }
    
    // 对应命令："确认支付"，事件："支付已确认"
    void confirmPayment() {
        if (status != OrderStatus::PAID) {
            throw DomainException("Only paid orders can be confirmed");
        }
        
        this->status = OrderStatus::CONFIRMED;
        this->updatedAt = std::chrono::system_clock::now();
        
        addDomainEvent(std::make_shared<PaymentConfirmedEvent>(orderId));
    }
    
    // 对应命令："发货订单"，事件："订单已发货"
    void ship(const std::string& trackingNumber) {
        if (status != OrderStatus::CONFIRMED) {
            throw DomainException("Only confirmed orders can be shipped");
        }
        
        if (trackingNumber.empty()) {
            throw DomainException("Tracking number is required");
        }
        
        this->trackingNumber = trackingNumber;
        this->status = OrderStatus::SHIPPED;
        this->updatedAt = std::chrono::system_clock::now();
        
        addDomainEvent(std::make_shared<OrderShippedEvent>(
            orderId, trackingNumber, shippingAddress));
    }
    
    // 对应命令："标记送达"，事件："订单已送达"
    void markAsDelivered() {
        if (status != OrderStatus::SHIPPED) {
            throw DomainException("Only shipped orders can be marked as delivered");
        }
        
        this->status = OrderStatus::DELIVERED;
        this->updatedAt = std::chrono::system_clock::now();
        
        addDomainEvent(std::make_shared<OrderDeliveredEvent>(orderId));
    }
    
    // 对应命令："确认收货"，事件："订单已完成"
    void complete() {
        if (status != OrderStatus::DELIVERED) {
            throw DomainException("Only delivered orders can be completed");
        }
        
        this->status = OrderStatus::COMPLETED;
        this->updatedAt = std::chrono::system_clock::now();
        
        addDomainEvent(std::make_shared<OrderCompletedEvent>(orderId, customerId));
    }
    
    // 对应命令："取消订单"，事件："订单已取消"
    void cancel(const std::string& reason, bool refundRequired = true) {
        // 业务规则：已发货的订单不能取消
        if (status == OrderStatus::SHIPPED || 
            status == OrderStatus::DELIVERED || 
            status == OrderStatus::COMPLETED) {
            throw DomainException("Cannot cancel order in current status");
        }
        
        this->status = OrderStatus::CANCELLED;
        this->cancellationInfo = {
            .reason = reason,
            .cancelledAt = std::chrono::system_clock::now(),
            .refundRequired = refundRequired && (status == OrderStatus::PAID || 
                                                status == OrderStatus::CONFIRMED)
        };
        this->updatedAt = std::chrono::system_clock::now();
        
        addDomainEvent(std::make_shared<OrderCancelledEvent>(
            orderId, reason, cancellationInfo.refundRequired));
    }
    
    // 对应命令："处理退款"，事件："退款已处理"
    void processRefund(const std::string& refundId, 
                      double refundAmount,
                      const std::string& reason) {
        if (status != OrderStatus::CANCELLED && 
            status != OrderStatus::REFUNDING) {
            throw DomainException("Only cancelled or refunding orders can be refunded");
        }
        
        // 业务规则：退款金额不能超过支付金额
        if (refundAmount > amount.getTotal()) {
            throw DomainException("Refund amount cannot exceed order total");
        }
        
        this->refundInfo = {
            .refundId = refundId,
            .refundAmount = refundAmount,
            .refundReason = reason,
            .refundedAt = std::chrono::system_clock::now()
        };
        this->status = OrderStatus::REFUNDED;
        this->updatedAt = std::chrono::system_clock::now();
        
        addDomainEvent(std::make_shared<RefundProcessedEvent>(
            orderId, refundId, refundAmount, reason));
    }
    
    // ==================== 查询方法 ====================
    bool canBeCancelled() const {
        return status == OrderStatus::CREATED || 
               status == OrderStatus::SUBMITTED ||
               status == OrderStatus::PAID ||
               status == OrderStatus::CONFIRMED;
    }
    
    bool canBeShipped() const {
        return status == OrderStatus::CONFIRMED;
    }
    
    bool requiresRefund() const {
        return cancellationInfo.refundRequired;
    }
    
    double getRefundableAmount() const {
        if (status == OrderStatus::PAID || status == OrderStatus::CONFIRMED) {
            return amount.getTotal();
        }
        return 0.0;
    }
    
private:
    void validateInitialState() {
        if (orderId.empty()) throw ValidationException("Order ID is required");
        if (customerId.empty()) throw ValidationException("Customer ID is required");
    }
    
    void validateShippingAddress(const std::string& address) {
        if (address.empty()) {
            throw ValidationException("Shipping address is required");
        }
        // 更详细的地址验证
    }
    
    void validatePaymentMethod(const std::string& method) {
        static const std::vector<std::string> validMethods = {
            "ALIPAY", "WECHAT_PAY", "BANK_CARD", "COD"
        };
        
        if (std::find(validMethods.begin(), validMethods.end(), method) == validMethods.end()) {
            throw ValidationException("Invalid payment method");
        }
    }
    
    void updateOrderAmount() {
        double subtotal = 0.0;
        for (const auto& item : items) {
            subtotal += item.calculateSubtotal();
        }
        
        // 计算运费策略（从业务规则识别）
        double shippingFee = calculateShippingFee(subtotal);
        
        // 应用订单级折扣（从"优惠券已应用"事件识别）
        double orderDiscount = calculateOrderDiscount();
        
        amount = OrderAmount(subtotal, shippingFee, orderDiscount);
    }
    
    double calculateShippingFee(double subtotal) {
        // 业务规则：满100包邮
        return subtotal >= 100.0 ? 0.0 : 10.0;
    }
    
    double calculateOrderDiscount() {
        // 这里可以调用优惠领域服务
        // 从"优惠券已应用"事件知道这里需要计算折扣
        return 0.0; // 简化实现
    }
```

```cpp
// 基于事件风暴流程设计的应用服务
namespace application::order {

class OrderApplicationService {
private:
    std::shared_ptr<domain::order::OrderRepository> orderRepository;
    std::shared_ptr<domain::inventory::InventoryService> inventoryService;
    std::shared_ptr<domain::payment::PaymentService> paymentService;
    std::shared_ptr<domain::logistics::LogisticsService> logisticsService;
    std::shared_ptr<EventPublisher> eventPublisher;
    
public:
    // 对应完整的订单创建流程（从事件风暴识别）
    CreateOrderResult createOrder(const CreateOrderCommand& command) {
        // 1. 验证命令（应用层验证）
        validateCommand(command);
        
        // 2. 生成订单ID
        std::string orderId = generateOrderId();
        
        try {
            // 3. 创建订单聚合（对应"订单已创建"事件）
            auto order = std::make_shared<domain::order::Order>(orderId, command.customerId);
            
            // 4. 添加商品（对应"订单商品已添加"事件）
            for (const auto& itemCmd : command.items) {
                domain::order::OrderItem item(
                    itemCmd.productId,
                    itemCmd.productName,
                    itemCmd.unitPrice,
                    itemCmd.quantity
                );
                order->addItem(item);
            }
            
            // 5. 提交订单（对应"订单已提交"事件）
            order->submit(command.shippingAddress, command.paymentMethod);
            
            // 6. 预留库存（对应"库存已预留"事件）
            auto reserveResult = inventoryService->reserveStock(
                orderId, extractProductQuantities(command.items));
            
            if (!reserveResult.success) {
                throw DomainException("Failed to reserve inventory: " + reserveResult.message);
            }
            
            // 7. 保存订单
            orderRepository->save(*order);
            
            // 8. 发布订单创建事件
            eventPublisher->publish(std::make_shared<domain::events::OrderCreatedEvent>(
                orderId, command.customerId, order->getTotalAmount()));
            
            return CreateOrderResult{
                .success = true,
                .orderId = orderId,
                .totalAmount = order->getTotalAmount(),
                .reservedUntil = reserveResult.reservedUntil
            };
            
        } catch (const DomainException& e) {
            // 处理领域异常
            return CreateOrderResult{
                .success = false,
                .errorCode = "DOMAIN_VALIDATION_FAILED",
                .errorMessage = e.what()
            };
        }
    }
    
    // 对应支付流程
    PaymentResult payOrder(const PayOrderCommand& command) {
        // 1. 获取订单
        auto order = orderRepository->findById(command.orderId);
        if (!order) {
            return PaymentResult{.success = false, .errorMessage = "Order not found"};
        }
        
        try {
            // 2. 调用支付服务（对应"订单已支付"事件）
            auto paymentResult = paymentService->processPayment(
                command.paymentMethod,
                command.orderId,
                order->getTotalAmount(),
                command.paymentDetails);
            
            if (!paymentResult.success) {
                return PaymentResult{
                    .success = false,
                    .errorCode = paymentResult.errorCode,
                    .errorMessage = paymentResult.errorMessage
                };
            }
            
            // 3. 更新订单状态为已支付
            order->markAsPaid(paymentResult.transactionId, paymentResult.paidAmount);
            
            // 4. 确认支付（对应"支付已确认"事件）
            order->confirmPayment();
            
            // 5. 扣减库存（对应"库存已扣减"事件）
            inventoryService->deductStockForOrder(command.orderId);
            
            // 6. 保存订单
            orderRepository->save(*order);
            
            // 7. 发布支付成功事件
            eventPublisher->publish(std::make_shared<domain::events::OrderPaidEvent>(
                command.orderId, 
                paymentResult.transactionId,
                paymentResult.paidAmount,
                command.paymentMethod));
            
            // 8. 触发发货流程（异步）
            eventPublisher->publish(std::make_shared<domain::events::OrderReadyToShipEvent>(
                command.orderId, order->getShippingAddress()));
            
            return PaymentResult{
                .success = true,
                .paymentId = paymentResult.paymentId,
                .transactionId = paymentResult.transactionId
            };
            
        } catch (const DomainException& e) {
            return PaymentResult{
                .success = false,
                .errorCode = "PAYMENT_FAILED",
                .errorMessage = e.what()
            };
        }
    }
    
    // 对应取消订单流程
    CancelOrderResult cancelOrder(const CancelOrderCommand& command) {
        auto order = orderRepository->findById(command.orderId);
        if (!order) {
            return CancelOrderResult{.success = false, .errorMessage = "Order not found"};
        }
        
        try {
            // 1. 取消订单（对应"订单已取消"事件）
            bool refundRequired = order->requiresRefund();
            order->cancel(command.reason, refundRequired);
            
            // 2. 如果已预留库存，释放库存
            if (order->isInventoryReserved()) {
                inventoryService->releaseStockForOrder(command.orderId);
            }
            
            // 3. 保存订单
            orderRepository->save(*order);
            
            // 4. 发布取消事件
            eventPublisher->publish(std::make_shared<domain::events::OrderCancelledEvent>(
                command.orderId, command.reason, refundRequired));
            
            // 5. 如果需要退款，触发退款流程
            if (refundRequired) {
                eventPublisher->publish(std::make_shared<domain::events::RefundRequestedEvent>(
                    command.orderId,
                    order->getRefundableAmount(),
                    command.reason));
            }
            
            return CancelOrderResult{
                .success = true,
                .orderId = command.orderId,
                .cancelledAt = std::chrono::system_clock::now()
            };
            
        } catch (const DomainException& e) {
            return CancelOrderResult{
                .success = false,
                .errorCode = "CANCELLATION_FAILED",
                .errorMessage = e.what()
            };
        }
    }
    
private:
    void validateCommand(const CreateOrderCommand& command) {
        if (command.customerId.empty()) {
            throw ValidationException("Customer ID is required");
        }
        if (command.items.empty()) {
            throw ValidationException("Order must contain at least one item");
        }
        if (command.shippingAddress.empty()) {
            throw ValidationException("Shipping address is required");
        }
        if (command.paymentMethod.empty()) {
            throw ValidationException("Payment method is required");
        }
    }
    
    std::string generateOrderId() {
        auto now = std::chrono::system_clock::now();
        auto timestamp = std::chrono::duration_cast<std::chrono::milliseconds>(
            now.time_since_epoch()).count();
        return "ORD" + std::to_string(timestamp) + std::to_string(rand() % 1000);
    }
    
    std::map<std::string, int> extractProductQuantities(
        const std::vector<CreateOrderItem>& items) {
        std::map<std::string, int> quantities;
        for (const auto& item : items) {
            quantities[item.productId] = item.quantity;
        }
        return quantities;
    }
};

} // namespace application::order
```

## 七、总结：事件风暴的价值
1. **统一语言**：业务和技术达成共识
2. **发现隐式需求**：识别出文档中没有的业务规则
3. **减少返工**：提前发现设计问题
4. **团队协作**：建立共同理解
5. **快速启动**：几天内完成复杂领域建模

**记住**：事件风暴不是一次性的活动，而是随着业务发展不断进行的迭代过程。建议每季度或每次重大需求变更时都进行一次轻量级的事件风暴。
