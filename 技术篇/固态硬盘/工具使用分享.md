# CrystalDiskInfo

| **属性名称** | **含义** | **如何判断是否正常** | **** |
| --- | --- | --- | --- |
| **读取错误率** | 读取数据时发生的错误比例 | ✅ 当前值200（满分）且原始值=0：优秀   ⚠️ 若原始值持续增加，预示磁头或盘片问题 | 企业级：几亿也在正常范围 |
| **起转用时** | 硬盘电机启动到就绪的时间 | ✅ 当前值161 > 临界值21：正常   ⚠️ 若接近临界值（21）或原始值>100ms，表明电机老化 |  |
| **启停次数** | 磁头加载/卸载循环次数 | ✅ 原始值0x40=**64次**：新盘正常   🚫 若单日增加>10次，可能电源管理异常 |  |
| **重新分配扇区数** | **关键指标！** 已替换的坏扇区数量 | ✅ 原始值=**0**：无坏道（最佳状态）   🔴 若>0立即备份数据！ |  |
| **寻道错误率** | 磁头定位错误率 | ✅ 当前值200且原始值=0：磁头健康   ⚠️ 若下降或原始值增加，磁头可能损伤 |  |
| **通电时间(小时)** | 累计工作时间 | ✅ 原始值0xA5C=**2652小时**（约110天）   🚫 >3万小时需警惕老化 |  |
| **起转重试次数** | 电机启动失败重试次数 | ✅ 原始值=0：无启动故障   ⚠️ >0表明供电或轴承问题 |  |
| **重新校准重试次数** | 磁头校准失败次数 | ✅ 原始值=0：校准正常   🔴 >0可能硬件故障 |  |
| **通电次数** | 硬盘加电次数 | ✅ 原始值0x14=**20次**：合理开关机   🚫 异常增加预示电源问题 |  |
| **加速度感应错误率** | 震动导致的错误 | ✅ 原始值5：轻微震动（可接受）   ⚠️ >20需检查硬盘固定 |  |
| **断电磁头缩回计数** | 意外断电时磁头紧急归位次数 | ✅ 原始值0xE=**14次**：正常范围   🚫 过高增加划伤盘片风险 |  |
| **磁头加载/卸载循环计数** | **关键指标！** 磁头伸缩机构寿命 | ✅ 原始值0xB08=**2824次**   🚫 接近设计极限（通常60万次）时需更换 |  |
| **温度** | 硬盘工作温度 | ✅ 原始值0x22=**34°C**（实际值）   ⚠️ >50°C需改善散热 |  |
| **扇区物理位置重分配事件计数** | 成功映射（05）+不成功映射（备份扇区无）+<font style="color:rgb(64, 64, 64);">硬盘自动修复临时错误（未触发重映射）</font> | ✅ 原始值=0：无坏道处理   🔴 >0表明坏道增加 |  |
| **有待处置扇区数** | **高危指标！** 待修复的疑似坏道 | ✅ 原始值=0：无风险   🔴 >0且持续增加预示坏道爆发 |  |
| **不可校正的扇区数** | **致命指标！** 永久损坏的扇区 | ✅ 原始值=0：无物理损坏   🔴 任何非零值都需立即更换硬盘 |  |
| **UltraDMA CRC 错误计数** | 数据线传输错误 | ✅ 原始值=0：连接稳定   ⚠️ >0需更换SATA线 |  |
| **写入错误率** | 写入数据失败率 | ✅ 当前值200且原始值=0：写入功能健康   ⚠️ 下降预示盘片或磁头问题 |  |
| **磁头飞行小时数** | 磁头工作总时长 | ✅ 原始值0x92=**146小时**   📊 应与通电时间比例一致 |  |
| **主机总计写入** | 生命周期总写入量 | ✅ 原始值0x1563C9149≈**5.76GB**：极低   💾 机械盘通常无写入寿命限制 |  |
| **主机总计读取** | 生命周期总读取量 | ✅ 原始值0x597C18DA≈**1.47GB**：极低   📈 监控异常读取增长 |  |

### **<font style="color:rgb(64, 64, 64);">用户操作指南</font>**
1. **<font style="color:rgb(64, 64, 64);">优先看原始值</font>**<font style="color:rgb(64, 64, 64);">：</font>
    - <font style="color:rgb(64, 64, 64);">关键属性（05/C5/C6）的原始值直接决定硬盘健康（必须=0）。</font>
2. **<font style="color:rgb(64, 64, 64);">对比当前值与临界值</font>**<font style="color:rgb(64, 64, 64);">：</font>
    - <font style="color:rgb(64, 64, 64);">若当前值持续下降，即使>临界值也预示老化。</font>
3. **<font style="color:rgb(64, 64, 64);">最差值辅助判断</font>**<font style="color:rgb(64, 64, 64);">：</font>
    - <font style="color:rgb(64, 64, 64);">若最差值曾暴跌，可能经历过异常断电或震动。</font>



### **3. 典型场景示例**
| **场景** | **待处置扇区数（C5）** | **重新分配扇区数（05）** | **重分配事件计数（C4）** | **说明** |
| --- | --- | --- | --- | --- |
| 发现1个新坏扇区，尚未处理 | +1 | 0 | 0 | 硬盘暂未尝试重映射，需下次写入确认。 |
| 成功重映射1个待处置扇区 | -1 | +1 | +1 | `C5` 转为 `05`，`C4` 记录事件。 |
| 重映射失败（备用区不足） | 不变（仍+1） | 0 | +1 | `C4` 增加，但 `05` 不变，`C5` 仍存在。 |
| 临时错误，硬盘自动修复 | -1 | 0 | 0 | 未触发重分配，仅 `C5` 减少。 |
