# **1****、基本操作命令**
## **1.1 ****安装****/****卸载****mysql**
1） mysqld --install （控制台要切换到mysqld.exe的目录下）

安装mysql数据库管理系统

2） mysqld --remove

卸载mysql数据库管理系统

## **1.2 ****启动****/****停止****mysql****服务**
1）net start mysql

启动mysql服务

2）net stop mysql

停止mysql服务

## **1.3 ****登入****/****登出数据库**
登入数据库服务器：

mysql -h地址 -u用户名 -p密码

1）mysql -h 192.168.10.155 -u root -p 123

向ip地址为192.168.10.155的主机的数据库服务器发出用户名为root，密码为123的登入请求

2）mysql -h loacalhost -u root -p 123

向本机的数据库服务器发起用户名为root，密码为123的登入请求.

3）mysql -u sgy -p 123

向本机的数据库服务器发起用户名为sgy，密码为123的登入请求。

登出数据库服务器：

1）quit

2）exit

## **1.4 ****查看所有数据库**
Show databases

登入数据库服务器后，可查看服务器下所有的数据库

## **1.5 ****切换到指定数据库**
use 数据库名

## **1.6 ****查看当前使用的数据库**
select database()

## **1.7 ****查看数据库下的所有表**
show tables from 数据库名

查看指定数据库下的所有表

show tables

查看当前数据库下的所有表

## **1.8 ****查看表的结构**
desc 表名

查看当前数据库下的指定表的结构

## **1.9 ****查看数据库系统变量**
show variables like ‘%character%’

显示数据库服务器中系统变量名有character字符串的所有变量

## **1.10 ****解释执行计划**
explain + sql语句

显示数据库服务器中系统变量名有character字符串的所有变量

## 1.11 导出数据库
export -u用户名 -p密码 数据库名>新建数据库的路径

# **2****、数据定义语言（****DDL****）**
对数据库、表格、字段的增删改

## **2.1 ****添加数据库**
CREATE DATABASE [IF NOT EXISTS] 数据库名 [CHARACTER SET 字符集名] [COLLATE 校对规则名];

1）create database db_sv

2）create database if not exists db_sv

如果不存在db_sv，则创建

3）create database db_sv character set utf8mb4 collate utf8mb4_general_ci

创建一个db_sv的数据库，字符集为utf8mb4，校队规则为utf8mb4_general_ci

## **2.2 ****删除数据库**
Drop database [if exists] 数据库名

1）drop database db_sv

2）drop database if exists db_sv

## **2.3 ****修改数据库**
1）alter database db_sv character set utf8mb4  

修改数据库数据集

2）alter database db_sv set db_defect           

修改数据库名称

## **2.4 ****添加表格**
create table 表名(字段名1  数据类型1  [约束1],字段名2  数据类型2  [约束2] )

1）create table sensor(id int primary key, name varchar)

## **2.5 ****删除表格**
drop table sensor

1）drop table if exists sensor

## **2.6 ****修改表格**
1）alter table sensor_old rename to sensor_new

修改表格名称

## **2.7 ****增加字段**
alter table 表名 add column 列名 类型

## **2.8 ****删除字段**
alter table 表名 drop column 列名

## **2.9 ****修改字段**
alter table 表名 change column 列名旧 列名新 类型

修改字段名

alter table 表名 modify column 列名 新类型 【新约束】

修改字段的类型或约束

## **3.1 ****单表查询**
select * from 表名 where 条件 group by 列名1 having 条件 order by 列名2 desc limit 0，5

单表查询的完整句子，执行顺序为from->where->group by->having->select->order by->desc->limit。

1）group by... having...

一般与分组函数（max，min，averge等）配合使用

count(*)

找出每个部门的最高薪资，并且显示薪酬大于1000的数据

Select deptNo,max(sal) from emp group by deptNo having max(sal)>1000

+--------+----------+

|   NULL |  5000.00 |

|      1 |  2450.00 |

|      2 |  3000.00 |

|      3 |  2850.00 |

+--------+----------+

2）select distinct 字段1，字段2 

distinct 表示如果有（字段1，字段2）相同则去掉记录，即去重

## **3.2 ****两表查询**
select * from 表1 join     表2 on 表1.字段1=表2.字段2

select * from 表1 left join  表2 on 表1.字段1=表2.字段2

select * from 表1 right join 表2 on 表1.字段1=表2.字段2

select * from 表1 full join  表2 on 表1.字段1=表2.字段2

## **3.3 ****三表及以上查询**
select * from 表1  join 表2 on 表1.字段1=表2.字段2 join 表3 on 条件

找出每一个员工的部门名称、工资等级和上级领导

Select

e.empName ‘员工’, d.deptName, s.grade, e1.empName ‘领导’

From

emp e

Join

	Dept d

on

e.deptNo = d.deptNo

Join

saleGrade s

on

e.sal between s.minSal and s.maxSal

left join

emp e1

on

e.mgr = e1.empNo;

## **3.4 ****子查询**
1）where后嵌套select

找出高于平均工资的员工信息

Select * from emp where sal > （select avg（sal）from emp）

2）from后嵌套select

找出每个部门平均薪资的等级

Select 

t.*,s.grade 

from (select deptNo,avg(sal) as avgsal from emp group by deptNo) t 

Join

saleGrade s

On

t.avgsal between s.minSal and s.maxSal

+--------+-------------+-------+

| deptNo | avgsal      | grade |

+--------+-------------+-------+

|      1 | 2450.000000 |     4 |

|      2 | 2258.333333 |     4 |

|      3 | 1737.500000 |     3 |

+--------+-------------+-------+

Emp e;

# **4、****数据操作语言（****DML****）**
## **4.1 ****添加行数据**
1)  Insert into 表名（列1，列2） values （valueA1，valueB1）,（valueA2，valueB2）

增加多行数据

2)  insert into 表名 values（value1，value2）

增加一行数据，注：必须所有列都赋值

3) insert into 表名 set 列名=values，列名=values

增加一行数据

4) insert into 表1（列名1，列名2）select 列名1，列名2 from 表2 

查找到的结果插入已有的表中

5) create table 表名（select * from 旧表名）   查找的结果插入新表中

6) insert into 表1（列名1，列名2）values（值1，值2) ON DUPLICATE KEY UPDATE 列名3=值3   

主键或者唯一索引有冲突只做更新，否则做插入

## **4.2 ****删除行数据**
delete from 表 where 条件

## **4.3 ****修改行数据**
update 表名 set 列名1=值1，列名2=值2 where 列名2=值2

修改满足条件的行数据

# **5、****数据控制语言（****DCL****）**
## **5.1 ****创建用户**
create user 用户名@ 地址  identified by 密码

1）create user ‘develop’ @ ‘127.0.0.1’ identified by 123

为本机的数据库服务器创建一个用户名为develop，密码为123的用户

## **5.2 ****删除用户**
1）Drop user ‘develop’ @ ‘151.168.1.25’

为ip为151.168.1.25主机的数据库服务器删除用户为develop的用户

## **5.3 ****修改用户**
1）set password = password(‘1234’)

修改当前的用户的密码为1234

2）set password for ‘sgy’@’localhost’ = password(‘1234’)

为sgy用户修改密码为1234

## **5.4 ****添加权限**
1）grant all on dept to ‘sgy’@’localhost’ 

授权sgy用户对数据库名为dept的所有操作

2）grant select, insert,update on dept to ‘sgy’@’localhost’ 

授权sgy用户对数据库名为dept的增改查操作

## **5.5 ****取消权限**
1）revoke all from dept to ‘sgy’@’localhost’ 

取消sgy用户对数据库名为dept的所有操作的授权
