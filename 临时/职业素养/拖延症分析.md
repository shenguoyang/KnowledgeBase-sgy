# 拖延症的5层分析 ——一篇文章让你对拖延症的认知上升到天花板


  


**零**

拖着不去做题，拖着不去改错，拖着不去背单词，拖着不去复习错题本，甚至，网上搜了一堆学习方法，拖着不去使用……

于是你想改变拖延——但是你拖着不去改……

网上讲拖延症的文章很多，全都很肤浅。这篇文章，将是你看到所有关于拖延症文章的天花板，它将终结拖延症这个话题。一旦你真正掌握了这个视频所讲的内核精髓，你再去看其他讲拖延症的文章，将会有一种一览众山小的俯视感，能看到所有其他文章的局限。

让我们由浅入深，逐级看破。


**壹**

如何解决拖延症？最浅的一层理解是：要提高执行力，坚决都去做，不能让自己犹豫！

所谓提高执行力，这是一句废话。执行力不高就是拖延症的另一个名词表述，换一个名词就能解决问题吗？显然不行。

类似的肤浅理解还有：要提高意志力！要有坚定的意志、毅力，去克服困难！注意了，有些家长和老师就喜欢用这种辞令去教训学生，这种理解和教育的方式都是错的，因为人正是因为意志、毅力用完了，所以才会拖延啊！这就相当于说，我这个题不会做怎么办？别人回答，那你就应该把它做出来！完全是废话。总之，用意志力去对抗拖延是行不通的。


在了解了上述解决方案不可行之后，部分人给出了第二级的方案：用远期收益和复利原理的认知教育，来解决拖延症。在这一层级上，他们对拖延症的理解是，拖延是由于认为自己的行动没有意义导致的。比如，认为自己今天就算把作业做完了成绩不会提高，就算把单词背了阅读也还是不会做，就算做了工作策划也不会升职加薪，所以干脆不做。

基于这样的理解，他们就试图教育人群，让人们了解，哪怕微小的改变也有巨大的意义，因为基于复利原理，每天变好1%，一年也会有37.78倍的增幅。

又或者用一些这种名言来说明道理：大多数人都高估了自己一年能做的事情，而低估了自己十年内能做的事。

![](https://cdn.nlark.com/yuque/0/2023/jpeg/26460094/1689331276697-7653b4ce-319c-49eb-a324-c0550c648a97.jpeg)

这样一通安慰以后，少部分人受到了鼓励暂时告别拖延，为了那个长期来看巨大的意义而开始行动，但剩下的人并不吃这一套，他们还是不明原因的拖延着：我知道高考很有意义，但我就是拖着不去学习；我知道考研很有意义，但我就是拖着不去背单词；我知道升职加薪很有意义，但我就是拖着不去做那个项目策划。

怎么办？


  


**贰**

当这样的说教式方法失效了以后，又有部分人给出了第三级的方案：用目标规划的相应技术去解决，比如smart法则之类的。

在这一层级上，他对拖延症的理解是：拖延是由于缺乏行动目标的清晰度导致的。因为你不能快速精确的知道自己要做什么，于是就陷入犹豫、停滞、拖延。基于这种理解，所以应该使用目标规划的相应技术解决，比如smart法则，就是帮你把目标做的特别的清晰。

S:Specific（具体的）

M:Measurable（可衡量的）

A:Attainable（可达到的）

R：Relevant（和其他目标相关的）

T：Time-bound（有截止日期的）


看起来挺唬人的样子，一度也非常流行，但经过大量人群实践后发现，这方法只对少部分人有用。对剩下的部分人，不论把目标怎么分解细化列成一张表都没用，他还是拖延着，甚至都连用做smart法则做规划表这件事都拖着不去做。更加无法开始做正式任务了。


由此又延伸出第四个层级的思考和解决方案。第四个层级的人认为，拖延症的问题，在于起始的那一瞬间最为困难，人们不愿意开始，人们是被任务难度吓倒了。基于这样的理解，他们又给出了一些简明的解决方案，比如5min法则，2min法则等。

5min法则是指，为了避免我们畏惧困难不去开始学习或工作，我们就在一开始哄骗自己：5分钟，只做5分钟的习题就行了！然后你的大脑就会想：5分钟并不可怕，我还是开始做题吧！


而2min法则是指，我们做事之前预判一下，如果一件事情2分钟之内能搞定，就不要堆积起来，而要立马做掉，因为如果各种小任务堆积的越来越多了，那我们就会更拖延。


这两个简单易行的法则你可以尝试一下，一开始还是有点用的，直到你的大脑意识到，5分钟法则只不过是自我欺骗而已，而我早就识破了这个骗局，于是不想动了。而2min法则一样效用有限，毕竟绝大部分学习工作任务都是要做几个小时甚至几天、几年了，2分钟连个零头都不算。

  


**叁**

至此为止，人类常用的拖延症相关解决方案都在这里了。我估计很多人都试过，大概率，没用。当一切方法都失效过后，拖延症，就成了绝症。


想要真正解决问题，我们需要进入，第5层。

如果不看这篇文章，你永远想不到还有第5层，更不知道第5层是什么样子，因为它已经脱离了人类意识表层的范畴，直接进入了潜意识这个大脑的黑箱区域。下面的案例和讲解将会给你的认知带来巨大的冲击，胆小者，现在就可以离开了。

留下来的人，带着开放的心态来看看这些案例吧。

一个学生A，由于拖延症，不愿意学习，高中成绩一塌糊涂，后来上了大专，想要发愤图强认真学习，但拖延症再次阻挡了他，不论他内心多么强烈的想要上进，都莫名其妙的无法开始学习。机缘巧合他找到了我，我对他做了个案进行了心理疏导，使用了一些方法与他的深层意识进行了沟通，发现他这么多年来学习差、拖延症、心神不宁，主要原因是他大概四五岁的时候曾经不小心看到了他父母发生性行为，受到了巨大的冲击，内心非常愧疚和恐惧，而他的父母不知如何处理这件事，只是让他走开。他的恐惧保留了下来，埋藏在大脑的最深层，不断向意识表层发射负面的力量，进而导致了各种行为问题——包括了他的拖延。

当我处理了他那次事件的遗留的愧疚和恐惧以后，他整个人立马变得开朗起来，后续的日子里也快速摆脱了拖延症的干扰。


下一个案例，一个高中女生B深陷拖延症，尤其是语文学科问题特别严重，各种阅读理解、作文的作业全都拖着不去做。各种办法全都用尽了，父母老师说教无数次，她自己也很痛苦，但就是改不了。当我跟她进行了个案沟通以后，顺着她潜意识的线索查询到，这一切源于她小学四五年级的时候遇到的一个障碍，她那时候的语文老师素质低下，莫名其妙的批评打压她，哪怕她当时语文成绩好也要批评她，说什么你以为你成绩好了不起啊，成绩好就不听老师的话吗？批评打压她的次数多了以后，给她留下了比较深的恐惧和愤怒，保留在大脑深处，多年以后到了高中，就以拖延、抗拒语文学习的形式保留了下来。当我安抚了她童年的恐惧和愤怒之后，女生的拖延也自动解除了。


再来一个案例，一个高中男生C，有严重的拖延症，虽然他知道写作业很重要，做错题本很重要，做结构化和模式识别的总结很重要，但是他就是拖着不去做，有一种剧烈的自我冲突，而他也不知道为什么会这样。在找我做个案的过程中，我挖掘出了他拖延的一个重要源头——他与父亲的关系不和，他小时候经常被父亲吼骂，要求他完全服从自己的命令，因为自己的命令都是为了他好。他感觉自己很委屈，父亲是错的，他的深层意识执着的想要跟父亲辩论，让父亲认错、向他道歉。而现实中父亲当然不会这样做，于是他的心智就一直卡在这里不得安宁，从而形成了学习时的各种拖延。当我改变了他的认知，化解了他和父亲的矛盾以后，他的拖延也改变了，更够认真学习了。


当你看懂了这些案例以后你就知道，为什么前4层的拖延症对治技术无法起效了，不仅必然性的无法治本，甚至连治标都很难了。不论是smart法则，2min或5min法则，还是那个长期收益复利原理的理论，怎么可能能够解决心智最底层关于各种原生家庭与童年经历的问题呢？而这些问题不解决，必然从心智最底层不断向表层散发出负面的力量形成包括拖延在内的多种问题。

而此处提到信念课，包括今天写的这整篇文章，是想要告诉你一个道理：不论是拖延，还是无法自律，缺乏毅力，假完美主义等，都和今天讲的拖延症的一样，是在心智当中嵌有多个层级的。其最底层的来源，都不是一些生硬而浮华的技术能解决的。

当然，这不是说1-4层的方法就完全无用，smart原则，2min或5min法则，以及复利理论长期主义等理念，也都有其相应的用途。但真正想解决问题的人来说，还是难免要开启一场向内的探索了。
