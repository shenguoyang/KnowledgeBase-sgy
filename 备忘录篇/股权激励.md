# 易思维（杭州）科技股份有限公司
**股权激励协议（限制性股权版）**


本《股权激励协议（限制性股权版）》（以下简称“**本协议**”）由以下双方于2023年10月20日在杭州市签署：

**甲方**：  **易思维（杭州）科技股份有限公司**(下称**“公司”**)

法定代表人：郭寅

**乙方**：**  **沈国阳

身份证号：_                  _

公司职务：_                  _

为本协议之目的，甲方、乙方合称“双方”，单称“一方”。

****

**鉴于：**

1、公司于2018年12月25日制定了《易思维（杭州）科技有限公司员工股权激励管理办法》（下称“**管理办法**”）并于后续相应修订；

2、经公司确认，乙方符合管理办法中针对激励对象的范围和条件，因此甲方拟向乙方授予一定数量的激励股权（下称“**本股权激励计划**”）。

**因此**，根据《中华人民共和国民法典》以及其他有关现行法律、法规的规定，各方就实施股权激励计划之事宜，经诚信、友好协商一致，自愿达成本协议如下：


**第一条****    ****定义**

1.1    “**本协议**”、“**甲方**”、“**乙方**”、“**公司**”具有本协议前言所规定的含义。

1.2    “**初始有限合伙人**”指公司实际控制人郭寅。

1.3    “**工作小组**”指公司董事会根据管理办法下设的股权激励计划日常管理工作小组。

1.4    “**集团公司**”指公司及其各自下属全资或控股子公司。

1.5    “**激励股权**”具有本协议第3.2.1条所规定的含义。

1.6    “**认购日**”具有本协议第3.3.3所规定的含义。

1.7    “**上市**”指集团公司或任一集团公司（1）在一个有声望的中国境内或境外证券交易所首次公开发行股票并上市；或者（2）作为一家上市公司重大资产重组交易中的标的公司且其股权被整体收购。

1.8    “**员工持股平台**”，截至本协议签署之日，指嘉兴易实至诚企业管理合伙企业(有限合伙)，一家根据中国法律成立并有效存续的有限合伙企业。不排除公司进一步设立其他实体作为员工持股平台实施本股权激励。


**第二条****    ****股权激励计划总体安排**

2.1      本协议签署生效后，由甲方授予乙方在一（1）年内自初始有限合伙人处以约定的行权价格（定义如下）通过员工持股平台受让公司部分股权的权利，乙方在符合本协议、公司上市计划及公司要求的情况下可以通过行使认购权而通过员工持股平台间接持有公司股权，并通过员工持股平台间接享有公司股东权益。

2.2      为避免异议，甲乙双方确认，本协议第四条项下内容仅适用于乙方届时已行使激励股权（即，乙方已通过行使激励股权而间接持有公司股权）之情形。


**第三条****    ****资格确认与自筹资金缴纳**

3.1      _乙方被授予、行使激励股权的资格_。乙方被授予激励股权应具备下述资格：

3.1.1        乙方归属于下述人员范围内：

a)     集团公司董事及高级管理人员；

b)     集团公司核心技术人员、中层管理人员及各部门做出杰出或特殊贡献的骨干员工；

c)      公司董事会认定的其他可参与计划人员。

3.1.2        乙方符合下述条件：

a)     认同公司理念；

b)     忠诚度高；

c)      为公司中长期发展做出持续的努力和贡献、或关键时期的重要作用；

d)     不在与公司存在业务竞争（潜在业务竞争）关系的任何企业里持股、任职、领薪，最近三年内不存在因违法违规行为被中国证监会或股转系统采取警告、处罚或其他负面措施；

e)     公司董事会认为乙方作为股权激励计划的激励对象应具备的其他条件。

3.2      _激励股权的授予_。

3.2.1        自本协议生效之日起，在满足本协议所约定之条件下，公司同意通过员工持股平台向乙方授予共计_叁仟捌佰捌拾玖元_人民币(RMB3889)的公司注册资本的认购权，前述公司股权所对应员工持股平台的财产份额为_贰万柒仟肆佰捌拾叁元_人民币(RMB27483)（“**激励股权**”）。

3.3      _激励股权的认购_。

3.3.1        乙方应当于本协议生效之日起一（1）年内或公司董事会决定的其他时间点向公司董事会/工作小组发出行权申请并行使激励股权的认购权。原则上，认购权应一次性行使，但公司董事会另有规定的除外；于任何情形下，乙方都不得向任何第三方转让其获得的认购权，或在该等认购权上设置任何担保性权益。

3.3.2        双方同意，针对激励股权中的员工持股平台财产份额，乙方合计应支付行权总额为_伍万壹仟捌佰陆拾元_人民币(51860)。

3.3.3        乙方应根据如下步骤行使激励股权：

a)     公司董事会作出决议同意初始有限合伙人与乙方就转让当期行权份额签署《财产份额转让协议》；

b)     公司董事会/工作小组向乙方发出缴款通知，乙方须在该等通知规定的期限内将激励股权对应的行权价格全部支付至初始有限合伙人指定的账户，乙方完成前述行权价格全额支付之日称为“**认购日**”；

c)      乙方与员工持股平台的普通合伙人、初始有限合伙人及其他员工持股平台中的有限合伙人（如需）签署《合伙协议》；

d)     乙方配合工作小组协助办理员工认购激励股权的工商变更手续。

3.3.4        为免疑义，如乙方未能按照公司董事会/工作小组的相关规定及时向初始有限合伙人支付行权价格的，均视为乙方自愿放弃行权。相应的激励股权对应的员工持股平台的财产份额由公司董事会另行决定处置方式。

3.4      乙方承诺将以自有的合法资金向初始有限合伙人购买激励股权。

**第四条****    ****退出与转让**

4.1      除非本股权激励办法另有明确约定，在员工持股平台的存续期间，由公司董事会/工作小组协助普通合伙人确定乙方退出或转让激励股权的相关事宜。

4.2      双方同意，若乙方在取得激励股权后离职，则乙方必须以相应退出价格将其在员工持股平台中所占的财产份额全部转让给普通合伙人确定的普通合伙人、初始有限合伙人、员工持股平台或普通合伙人指定的第三人。

4.3      双方同意，自乙方取得激励股权起，乙方应当遵守下列锁定期要求：

4.3.1        若公司在管理办法生效之日起五（5）年内没有启动上市计划，则对于每次认购的激励股权，自认购日起满五（5）年；

4.3.2        若公司在管理办法生效之日起五（5）年内正式启动上市计划，则锁定期自认购日至按照中国证监会规定的禁止相关人员出售公司股票的期限或员工持股平台届时有效承诺的不出售其在公司股权的期限（以较晚者为准）；

4.3.3        尽管有上述第4.3.1条规定，若本股权激励计划实施五（5）年后的任意时间，公司启动上市的，为避免对公司上市审批造成影响，则自公司与券商签订保荐与承销协议或类似协议之日（以下简称“**上市启动日**”），就乙方尚未处置的员工激励平台的份额自动锁定，直至前述第（2）项约定的锁定期结束。

4.4      在锁定期内，未经员工持股平台普通合伙人的书面同意，乙方不得转让、抵押、质押、赠与或以任何方式处置其间接持有的激励股权。若激励对象在取得激励股权后离职，则该激励对象必须以相应退出价格将其在员工持股平台中所占的财产份额全部转让给普通合伙人确定的普通合伙人、初始有限合伙人、员工持股平台或普通合伙人指定的第三人。

4.5      经普通合伙人书面同意，在锁定期内乙方拟处置激励股权或因离职而必须转让激励股权的，应当以相应退出价格将其在员工持股平台中所占的财产份额全部转让给普通合伙人确定的普通合伙人、初始有限合伙人、员工持股平台或普通合伙人指定的第三人，退出节点按照乙方向公司提出处置请求或者离职之日计算。前述退出价格具体指：

4.5.1        乙方主动或被动离职，或主动申请处置激励股权，且未有不当行为时：

退出价格 = 乙方为取得激励股权实际支付的价格 + 按照10%（单利）自认购日至乙方根据本项之规定收到退出价格之日计算的利息

4.5.2        乙方主动或被动离职，或主动申请处置激励股权，且存在一种或多种不当行为时：

退出价格 = 乙方为取得激励股权实际支付的价格

4.5.3        前述“不当行为”是指：

a)     乙方严重违反适用于公司的任何法律、法规或公司章程；

b)     乙方从事任何违法行为，且受到刑事处罚；

c)      乙方有不忠诚于公司的行为，包括但不限于从公司辞职并受雇于与公司业务有直接或间接竞争的其他公司或实体，或从与公司的关联交易中获得利益（但事先向公司披露并经公司董事会批准的除外）；

d)     乙方实质违反其与公司之间的任何协议，包括但不限于泄露公司商业秘密等保密信息，实质上没有履行或拒绝履行作为公司员工或董事的义务（但因被授权人死亡或丧失劳动能力的除外）；

e)     乙方违反公司任何规章制度并给公司的财产、声誉或其他员工或董事造成损失、损害或伤害；

f)       乙方有其他任何对公司业务、声誉或财务状况造成严重不良影响的行为。

尽管有上述规定，工作小组有权自行判断是否书面豁免乙方的上述该等不当行为。

4.6      锁定期届满后，乙方拟处置激励股权或因离职而必须转让激励股权的，应当以相应退出价格将其在员工持股平台中所占的财产份额全部转让给普通合伙人、初始有限合伙人、员工持股平台或普通合伙人指定的第三人，退出节点按照乙方向公司提出处置请求或者离职之日计算：

4.6.1        倘若届时公司尚未实现上市，乙方可参照该等财产份额对应的公司股权届时的公允价值作为退出价格向第三方转让（前提是转让给该等第三方不会对公司上市构成不利影响），但在同等条件下，普通合伙人或其指定方享有优先购买权；

4.6.2        倘若届时公司已经实现上市计划，根据市场情况，公司将尽力为员工持股平台安排不少于每年三次的公司股权减持，乙方如需减持公司股权的，应积极配合相关计划。在该等情形下，退出价格按照届时市场实际交易情况而定，员工持股平台取得减持对价（扣除员工持股平台管理成本、因股票出售而需承担的交易费用和税费后）按相对持股比例向要求减持的激励对象予以分配。

4.7      乙方因任何原因退出本股权激励计划，均应遵照相关规定经公司董事会审定，由工作小组统一办理转让手续，凡乙方违反规定擅自直接或间接转让的，乙方就激励股权处置所得全部收益应当归属于普通合伙人。


**第五条****    ****甲方的权利和义务**

5.1      应建立并保管《内部员工激励名册》，作为乙方行权、出资支付对价的书面凭证，并载明下列事项：

(一)        乙方的姓名、性别、年龄、身份证号、住址、电话、邮编、所在公司部门、职务、继承人/承受权利人及收款凭证编号；

(二)        乙方所持有激励股权的变动情况，包括变动日期、数量、价格、收付金额等；

(三)        乙方所获激励股权收益的分配情况，包括年度、分配额度等；

(四)        乙方及经办人签章。

5.2      乙方不再持有激励股权的，甲方应及时协助办理员工持股平台变更登记手续，并在《内部员工激励名册》中进行相应标注。

****

**第六条****    ****乙方权利与义务**

6.1      倘若乙方决定行使激励股权对应的认购权，则乙方应当按照本协议的规定及时向公司董事会/工作小组送达行权申请。

6.2      乙方应当按照本协议的规定按期足额支付行权价格。

6.3      乙方应当及时向甲方提供为办理员工持股平台变更登记所需准备的工商文件，并协助甲方和/或员工持股平台在主管政府部门完成相关的登记/变更登记手续。

6.4      乙方在办理激励股权的工商登记而成为员工持股平台的有限合伙人之前，不具有员工持股平台的有限合伙人资格，不享有有限合伙人权利亦不承担有限合伙人义务。在相关工商设立或转让登记事项完成后，乙方成为员工持股平台的有限合伙人，即甲方间接股东，但其仅享有对其所间接持有的激励股权处分后的经济收益权，不享有员工持股平台的投票权等权利。乙方需将全部投票权委托普通合伙人行使。

**__**


**第八条****    ****相关税费**

8.1      乙方应根据中国法律之规定，对其因参与本股权激励计划或行使本股权激励计划项下的权利而取得的任何形式的收益，向相关税务机关缴纳个人所得税。

8.2      员工持股平台中的管理成本，应由包含乙方在内的全体合伙人承担。在分配红利/处置收益时，管理成本将在红利中先期扣除，剩余所得再分配给乙方。


**第九条****    ****免责条款**

9.1      在本协议的执行过程中，因国家的法律、政策等发生重大变更、乙方违反本协议约定或甲方关于股权激励计划的规章制度，导致甲方无法履行本协议，甲方不承担责任。


**第十条****    ****协议与聘用关系**

10.1  甲方与乙方签署本协议不构成甲方对乙方聘用期限和聘用关系的任何承诺，甲方对乙方的聘用关系仍按甲方和乙方签署的劳动合同的有关约定执行。


**第十一条****          ****争议的解决**

11.1  甲、乙双方在本协议执行过程中发生的争议，双方应首先通过友好协商解决。若在任何一方提议协商之日后三十（30）日内不能通过协商解决前述争议的，则任何一方均可将该争议提交至上海国际仲裁中心依照其届时有效的仲裁规则在上海进行仲裁。仲裁裁决是终局的，对双方均具约束力。


**第十二条****          ****协议的终止**

12.1  倘若乙方违反本协议有关约定、违反甲方股权激励计划或者国家法律政策，甲方有权视具体情况通知乙方终止本协议且甲方不需承担任何法律责任。
